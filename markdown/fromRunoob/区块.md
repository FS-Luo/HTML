# Markdown 区块
Markdown 区块引用是在段落开头使用 > 符号，然后后面紧跟一个 空格 符号。

> 区块引用  
菜鸟教程  
学的不仅是技术更是梦想

另外区块是可以嵌套的，一个 > 是最外层，两个 > 是第一层嵌套，以此类推。

> 最外层
>> 第一层嵌套  
>>> 第二层嵌套 

### 区块中使用列表
> 区块中使用列表  
> 1. 第一项  
> 2. 第二项
> + 元素一  
> + 元素二

### 列表中使用区块
如果要在列表项目放进区块，那么就需要在 > 前添加四个空格的缩进。
* 第一项
    > 菜鸟教程
    > 学的不仅是技术更是梦想
* 第二项
